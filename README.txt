Скейтч позволяет собрать фотосинхронизатор, отправляющий ИК сигнал при попадании света на датчик или при нажатии кнопки
Использована плата Arduino Nano v3 (LGT8F328p, совместимая)

Подключение:
D3 - ИК передатчик 38кГц
D8 - кнопка с подтяжкой на землю через резистор около 10 кОм (Съемка)
D7 - датчик света стандартный цифровой, выдает LOW при срабатывании

Библиотеки:
1) Библиотека irremote
